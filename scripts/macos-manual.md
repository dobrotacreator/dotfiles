# macOS Manual Setup

## Before Bootstrap

- Install Command Line Tools or Xcode.
- Install Homebrew from https://brew.sh.
- Clone dotfiles to `~/dotfiles`.
- Run `~/dotfiles/scripts/bootstrap`.

## Apple / System

- Sign in to Apple Account.
- Enable Find My Mac.
- Configure Touch ID.
- Configure FileVault.
- Configure iCloud Drive/Desktop/Documents manually.
- Review battery, display, Lock Screen, and privacy settings.
